element selector : html tag를 선택하는 선택자, 같은 태그 전부 선택함

id selector : 특정 id를 가진 애만 적용하는 선택자, 스타일 쪽에서는 id에 #을 붙여줘야함

id는 html element를 고유하게 지칭, 하나의 html 문서 안에서 id 중복은 불가능

class selector : 같은 css를 적용하고 싶은 부분들을 하나의 반으로 편성

css에서 클래스는 클래스 명 앞에 .으로 적용 class는 이렇게 적용가능
